## Methodology  

1. **Data Modeling**  
   - Created `RETAIL_SALES` database with properly defined schema  
   - Applied normalization principles to ensure referential integrity  

2. **Data Cleaning & Transformation**  
   - Removed nulls and duplicates  
   - Validated data types and transaction consistency  

3. **SQL Analysis**  
   - Queried for KPIs such as **total sales**, **average customer age**, **category trends**, and **shift-based transactions**  
   - Conducted year-over-year and gender-based comparative analysis  

4. **Power BI Visualization**  
   - Built interactive dashboards with dynamic filters  
   - Developed calculated columns for time-series grouping (month, year)  
   - Integrated DAX measures for KPI computation
